## iPhone / iPad Setup Steps

You can configure the repository from mobile Safari or the GitHub mobile app if you are signed in.

Mobile Safari:

1. Open `https://github.com/Sena-lilly/codex-app-workflow`.
2. Tap the repository settings or edit/About controls. If the mobile layout hides them, request the desktop website from Safari.
3. Edit the About description.
4. Leave Website empty.
5. Add the suggested topics one by one.
6. Save changes.
7. Confirm the About sidebar shows the description and topics.

GitHub mobile app:

1. Open the repository.
2. Open repository settings or repository details.
3. Edit description and topics if the app exposes those fields.
4. If topic editing is unavailable, use mobile Safari with desktop website mode.
